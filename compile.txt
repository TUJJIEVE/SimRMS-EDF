
Readme for execution of the program

1  Too execute use g++ compiler first open the terminal using ctrl+alt+t then type the below commands.
2. First unzip by using "unzip filename.zip" command
3. Then change the directory using cd in the terminal
4. Then type "g++ filename.cpp" to compile the program.Make sure that the input file is in the same directory as the cpp files.
5. To execute type "./a.out"
